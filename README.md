# SYNOPSIS

Manages a linux user.

# INSTALL

    $ sparrow plg install user

# USAGE

    $ sparrow plg run user --param name=sparrow
    $ sparrow plg run user --param name=sparrow --param action=delete

# Parameters

## name

User name.
  
## action

One of two: create|delete. Default value is `create`.
